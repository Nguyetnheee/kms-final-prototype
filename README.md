# UAV KMS – Next.js

Ứng dụng được xây dựng hoàn toàn bằng Next.js App Router và JavaScript. Giao diện được tạo bởi React component và CSS, không dùng iframe hoặc các trang HTML tĩnh.

Chạy dự án:

```bash
npm install
npm run dev
```
